## 👍 3. Rule
- 매일 가벼운 인사로 시작합니다.
- 1일 1커밋을 합니다.
- 코드리뷰를 받고 싶을 때 PR 시에 전반적인 설명을 적습니다.
- 코드리뷰는 1000 라인을 넘을 수 없습니다.
- 코드리뷰는 존중하는 마음으로 합니다.
- 개인의 코딩 스타일을 존중합니다.
- 네이밍에 대한 조언은 함수가 어떤 역할을 수행하는지 뚜렷하지 않을 떄 합니다.
- 코드리뷰는 실력이 좋은 사람이 하는 것이 아닙니다.
